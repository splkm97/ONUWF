# ONUWF
One Night Ultimate Werewolf ver 2

# 실행시 설치해야하는 go package

- mongoDB
```
go get go.mongodb.org/mongo-driver/mongo
```

- discordgo
```
go get github.com/bwmarrin/discordgo
```

- discordgo embed
```
go get github.com/clinet/discordgo-embed
```
